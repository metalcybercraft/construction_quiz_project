Проект "Разработка Web-приложения на Django" на тему:
"Веб-приложение на Django для тестирования знаний строительных терминов"

Название проекта:"Construction Quiz"


**Как запустить проект?**

1. Клонируйте репозиторий:
2. Перейдите в папку: `cd construction_quiz`
3. Создайте виртуальное окружение: `python -m venv venv`
4. Активируйте вирутуальное окружение (пунк 3)
5. Установите зависимости: `pip install -r requirements.txt`
6. Выполните миграции: `python manage.py migrate`
7. Запустите сервер: `python manage.py runserver`

**Что может WEB-приложение?**

CRUD для вопросов (создание, чтение, редактирование)
Поиск по тексту вопроса
Случайный тест из 5 вопросов с прогресс-баром и пояснениями
Серо-белый интерфейс на чистом CSS
